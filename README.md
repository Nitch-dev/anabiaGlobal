# Anabia Global Website

This is a complete static website for your overseas employment agency.

## How to open

1. Open `index.html` in your browser.
2. For best behavior, use VS Code Live Server extension or any local server.

## Replace with your real assets

### CEO photo
- Replace file: `assets/images/ceo-placeholder.svg`
- Recommended size: 900 x 620 px
- Keep file name the same, or update image path in `index.html`

### Demand letters
- Replace files:
  - `assets/images/demand-1.svg`
  - `assets/images/demand-2.svg`
  - `assets/images/demand-3.svg`
- You can use JPG, PNG, or PDF previews and update links in `index.html`

### Certificates
- Replace files:
  - `assets/images/certificate-1.svg`
  - `assets/images/certificate-2.svg`
  - `assets/images/certificate-3.svg`

### Agency profile PDF
- Current PDF path: `assets/docs/Anabia-OEP-Profile.pdf`
- If you replace it, keep the same file name or update the certificate section link in `index.html`

## Important text to update

Edit in `index.html`:
- Office address
- Phone number
- Email
- Countries list
- Jobs and demand letter details

## Form behavior

The Apply form currently shows a success message on submit.
To receive real submissions, connect it to:
- Formspree
- Netlify Forms
- A custom backend endpoint
